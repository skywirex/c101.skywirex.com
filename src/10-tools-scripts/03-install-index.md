# Tổng hợp hướng dẫn cài đặt tools

Trang này dùng để gom tất cả hướng dẫn cài đặt công cụ, giúp dễ mở rộng về sau.

## Danh sách hướng dẫn

- [Cài đặt Yazi trên Debian 12](./02-cai-dat-yazi-debian12.md)

## Gợi ý mở rộng

Khi thêm tool mới, tạo file theo mẫu:

- `04-cai-dat-<ten-tool>.md`
- `05-cai-dat-<ten-tool>.md`

Sau đó bổ sung link vào mục **Danh sách hướng dẫn** ở trang này.
