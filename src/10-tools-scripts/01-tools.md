# Tools

Xem tổng hợp hướng dẫn cài đặt tại: [Install Index](./03-install-index.md).

## [VMware Workstation Pro](https://www.vmware.com/products/desktop-hypervisor/workstation-and-fusion)

Phần mềm ảo hóa desktop mạnh, hỗ trợ tạo và quản lý nhiều máy ảo trên Windows/Linux. Phù hợp để lab, test hệ điều hành, snapshot và rollback nhanh.

![VMware Workstation Pro (For Windows) 25H2](https://pub-b731809282d4443bba205fbf4c8ae4ee.r2.dev/18a953377febba0d3e1c9bb809d4667d.png)

## [Proxifier](https://www.proxifier.com/)

Công cụ ép ứng dụng đi qua proxy (SOCKS/HTTPS) ngay cả khi app không hỗ trợ proxy native. Hữu ích khi route traffic theo rule, bypass theo app/process.

## [Yazi](https://yazi-rs.github.io/)

Terminal file manager hiện đại, nhanh, hỗ trợ preview tốt và thao tác bàn phím tối ưu cho workflow CLI.

Hướng dẫn cài đặt: xem tại [Cài đặt Yazi trên Debian 12](./02-cai-dat-yazi-debian12.md).

## [Chromium trên VPS](https://share.note.sx/6mxc433m#RGe0vZnoJmUkB1RKCHii52KpzT6hrrLTo2iU+4cRCtc)

Thiết lập Chromium chạy trên VPS để truy cập/tự động hóa tác vụ web từ môi trường server.
